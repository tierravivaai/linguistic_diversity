# Language Endangerment Report — CBD Parties Only

## Overview

- **Total languages in dataset:** 8,126
- **Languages with coordinates:** 7,935
- **Languages with country data:** 8,122
- **Endangered languages** (shifting, threatened, moribund, or nearly extinct): 4,108 (50.6%)

This report covers only languages spoken in CBD Party countries (non-Parties such as the Holy See and the United States are excluded).

Note: A language spoken in multiple countries is counted once per country. Therefore country-level totals sum to more than the total number of unique languages.
